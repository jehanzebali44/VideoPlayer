# 🎬 Video Player

A modern and powerful Android Video Player built with **Kotlin** and **Jetpack Compose**. Designed to deliver a smooth, lightweight, and feature-rich media playback experience with advanced controls, gesture support, playlists, subtitles, and audio playback capabilities.

---

## ✨ Features

### 🎥 Video Playback

- Play all popular video formats
- Hardware accelerated playback
- Smooth and high-performance video rendering
- Background playback support
- Resume playback from last position
- Auto-rotate video orientation

### 🎵 Audio Mode

- Play videos as audio (MP3 mode)
- Background audio playback
- Continue listening with screen off
- Media notification controls

### 📂 Library Management

- Automatic video scanning
- Browse videos by folders
- Search videos instantly
- Sort videos by name, size, date, and duration
- Recently played videos

### ❤️ Favorites & Playlists

- Add videos to favorites
- Create and manage playlists
- Edit and manage playlists
- Shuffle playlist playback
- Play next and previous videos

### 📝 Subtitle Support

- Built-in subtitle support
- Load external subtitle files
- Subtitle synchronization
- Enable or disable subtitles

### 🎚️ Audio Enhancements

- Built-in Equalizer
- Bass Boost
- Audio Presets
- Volume Enhancement

### 👆 Gesture Controls

- Brightness gestures
- Volume gestures
- Swipe to seek
- Double tap seek forward/backward
- Pinch to zoom

### ⚙️ Playback Controls

- Playback speed control
- Screen lock
- Repeat modes
- Shuffle mode
- Sleep timer
- Auto screen rotation

### 🖼️ Picture in Picture

- Floating player support
- Continue watching while using other apps

### 🌙 User Experience

- Modern Material Design
- Jetpack Compose UI
- Dark Mode support
- Responsive layouts
- Lightweight and fast

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Android Media3 (ExoPlayer)
- MVVM Architecture
- Coroutines
- Room Database
- DataStore
- Navigation Compose
